# Cloud-Kitchen-java
Cloud Kitchen Management System developed in Java using Object-Oriented Programming concepts. The project supports menu management, order processing, and billing, simulating real-world food delivery backend systems.
# 🍔 Cloud Kitchen Management System

## 📌 Overview
A Java-based application to manage menu, orders, and billing for a cloud kitchen.

## 🚀 Features
- Add/View Menu
- Place Orders
- Billing System

## 🛠️ Tech Used
- Java
- OOP Concepts

## ▶️ Run
Compile:
javac *.java

Run:
java Main
